## Title: Personal Manufacturing Execution System

## Description:
This is a full-stack Manufacturing Execution System (MES) application built with Java and Spring Boot. The application allows users to define and execute custom step-by-step processes, managing data for each method and its associated steps. It demonstrates layered architecture, dynamic server-side rendering, entity relational mapping, and integration of business logic through a service layer.

## Features
- Backend built with Java and the Spring Boot framework
- Controllers used for request routing and rendering dynamic views (Thymeleaf)
- Service layer handles business logic and data transformations
- Repository layer manages data persistence with Spring Data JPA and H2 database
- Method and MethodStep entities designed with JPA, including relational mapping
- Frontend dynamically rendered using HTML, CSS, and Thymeleaf templates
- Step-by-step execution of processes, allowing users to track and manage each method

## Tech Stack
- **Backend:** Java, Spring Boot  
- **Frontend:** HTML, CSS, Thymeleaf  
- **Database:** H2, Spring Data JPA  
- **Architecture:** Layered MVC (Controller, Service, Repository)
